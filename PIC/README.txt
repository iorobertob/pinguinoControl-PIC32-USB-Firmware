These are the firmware files to load into the Pinguino PIC32 OTG board. 

They are programmed and compiled with Pinguino's own IDE. 

Find it on Pinguino.cc

The .pde file is the main file to load in Pinguino's IDE.

puredaga.c is an include file that must be added to Pinguino's include's folder.

puredata.pdl32 are definitions that are used in the IDE. This file must be added in the pdl32 files folder of Pinguinos's IDE.
